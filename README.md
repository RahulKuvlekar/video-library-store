# Video Library Store

Video Library Store is a video library for Sports built using React, HTML, CSS & Mockbee .

## Features Present

- Authentication
  - Login
  - Logout
  - Signup
- Video Listing
- Filter Products by Category
- Playlist Management
- Likes / Dislikes
- History
- Watch later
- Search by video title
- Single Video Page
- Notes on a video
- Loading & Alerts
